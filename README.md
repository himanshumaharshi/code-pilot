## :toolbox: Getting Started

### :gear: Installation

Clone the repository
```bash
git clone <repository-url>
```
Install dependencies
```bash
npm install
```
Set up the following
```bash
online code compiler api
```
Run the application
```bash
npm run dev
```


### :triangular_flag_on_post: Deployment

Build Command
```bash
$ npm run build
```
